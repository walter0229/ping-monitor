# EXANET Ping & Traceroute Monitor

Professional real-time network monitoring tool.

## Features
- Real-time Ping statistics (Sent, Received, Loss, Latency)
- Parallel Traceroute with hop-by-hop ping updates
- Interactive Chart.js visualization
- IP Location Data and Google Maps integration
- Custom IP management (Edit/Delete via context menu)

## Deployment
This app is designed to be deployed on platforms like **Render**, **Railway**, or any server supporting Python/FastAPI.

### Local Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `python app.py`
3. Open `http://localhost:8000`
